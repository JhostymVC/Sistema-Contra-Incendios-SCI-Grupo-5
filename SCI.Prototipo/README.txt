Este proyecto (SCI.Prototipo) es un prototipo simplificado del Sistema Contra Incendios (SCI).

Propósito:
- Probar rápidamente la interfaz de consola y la lógica básica de sensores sin afectar el proyecto principal.
- Servir como base para iteraciones de diseño del sistema desde cero, mostrando un flujo simplificado.

Instrucciones:
1. Abra la solución en Visual Studio.
2. Establezca SCI.Prototipo como proyecto de inicio para ejecutar la demo del prototipo.
